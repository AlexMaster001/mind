# 🌿 Тихая поддержка

Веб-приложение для психологической поддержки детей, подростков и взрослых.  
Позволяет анонимно пройти тесты, получить интерпретацию, сохранить результаты и получить персонализированное наблюдение от специалиста.

🔗 **Демо**: [https://mind-orcin-seven.vercel.app](https://mind-orcin-seven.vercel.app)

---

## 📌 Описание

Проект разработан как дипломная работа по специальности **09.02.07 «Информационные системы и программирование»** в ЧПОУ «Московский городской открытый колледж».

Основные цели:
- Снизить порог обращения за психологической помощью.
- Предоставить инструмент первичной диагностики (тревожность, выгорание, детско-родительские отношения).
- Поддержать родителей через образовательные материалы и кружок «Свободный кубик».
- Создать безопасное, анонимное и доступное цифровое пространство.

---

## 🧩 Функционал

### Для всех пользователей:
- Анонимное прохождение трёх тестов:
  - **GAD-7** — уровень тревожности,
  - **CBI** — эмоциональное выгорание,
  - **EMBU-S** — восприятие родительского поведения.
- Просмотр раздела «Для родителей».

### Для зарегистрированных пользователей:
- Сохранение результатов тестов в личном кабинете.
- Управление данными о детях (имя, возраст).
- Запись в кружок «Свободный кубик» (3–12 лет).
- Получение индивидуальных наблюдений от психолога.

### Для администратора:
- Отправка, редактирование и удаление наблюдений.
- Просмотр списка пользователей и их детей.
